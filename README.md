# DevBetterWeb

A simple web application for devBetter.com.

## Features

- Register
- Login
- View Public Questions/Topics
- Validate Accounts via Email (SendGrid)

### Members Only

- Update Profile
- View Member List
- View Recorded Coaching Sessions

### Administrators Only

- View Users
- View Roles
- Add/Remove Users to Role
- Add/Remove Roles to User
